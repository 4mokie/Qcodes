Station¶

Station objects - collect all the equipment you use to do an experiment.

class qcodes.station.Station(*components, *, monitor=None, default=True, update_snapshot=True, **kwargs)[source]¶

A representation of the entire physical setup.

Lists all the connected Components`s and the current default measurement (a list of actions). Contains a convenience method `.measure() to measure these defaults right now, but this is separate from the code used by Loop.

add_component(component, name=None, update_snapshot=True)[source]¶

Record one component as part of this Station.

Returns the name assigned this component, which may have been changed to make it unique among previously added components.

measure(*actions)[source]¶

Measure the default measurement, or parameters in *actions.

set_measurement(*actions)[source]¶

Save a set *actions as the default measurement for this Station.

These actions will be executed by default by a Loop if this is the default Station, and any measurements among them can be done once by .measure
