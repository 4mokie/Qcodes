qcodes.BreakIf¶

class qcodes.BreakIf(condition)[source]¶

Loop action that breaks out of the loop if a condition is truthy.

Parameters:condition (callable) – a callable taking no arguments. Can be a simple function that returns truthy when it’s time to quit May also be constructed by deferred operations on Parameter.
Raises:TypeError – if condition is not a callable with no aguments.

Examples

>>> BreakIf(gates.chan1 >= 3)
>>> BreakIf(abs(source.I * source.V) >= source.power_limit.get_latest)
