qcodes.utils.deferred_operations¶

Classes

class qcodes.utils.deferred_operations.DeferredOperations(call_func, args=(), call_parts=())[source]¶

Bases: object

NOTE: This class is DEPRECATED and you should not use it. You can achieve the same functionality using good ole lambdas.

Make math and logic operations return callables to defer execution of arbitrary formulae.

Can be used as a mixin class - in which case you should NOT call __init__ and you SHOULD override __call__ and get with identical results.

You can evaluate a DeferredOperations by calling it or by calling its .get() method

Parameters:
  • call_func – a callable taking no arguments
  • call_parts – the original callables, in case we want to record them all (ie raw data)

Examples

>>> d = DeferredOperations(lambda:42)
>>> d()
42
>>> (d*5)()
210
>>> (d>10)()
True
>>> ((84/d) + (d*d))()
1766

Note

and & or are special: there are no magic methods for the regular and & or, so we need to use the bitwise operators &,| as boolean operators. We DO NOT short-circuit them; the right side is always evaluated.
